- 电脑访问互联网某台服务器的IP地址时，是怎么知道它的MAC地址的？（ARP/代理ARP协议） 
- 电脑访问互联网某台服务器时，通信数据包如何将其从局域网发送到互联网？（NAT地址翻转）[参考](http://www.52im.net/thread-3506-1-1.html?spm=a2c6h.12873639.article-detail.16.570f3ab0TTsNAj)
